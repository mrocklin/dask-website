Dask

Parallel Computing in Python

Programming Paradigms

Arrays: Scalable NumPy arrays

Dask arrays coordinate many Numpy arrays in a grid either on disk on your laptop, or in distributed memory across your cluster.

import dask.array as da
x = da.random.random((10000, 10000), chunks=(1000, 1000))
y = x.dot(x.T + 1) - x.mean(axis=0)

Applications: Climate and earth science, biological and medical imaging, optimization algorithms

Dataframes: Scalable Pandas dataframes

Dask dataframes coordinate many Pandas dataframes along an index, either on disk on your laptop, or in distributed memory across your cluster.

import dask.dataframe as dd
df = dd.read_parquet('s3://bucket/*.parq')
df.groupby(df.name).amount.mean()

Applications: Data analysis, time series, ...

Machine Learning: Scalable algorithms for Scikit-Learn APIs 

from dask_ml.models import LogisticRegression
est = LogisticRegression(...)
est.fit(train, test)

Applications: Regression, classification, pipelines

Getting Started

Install and deploy Dask on your laptop or cluster

  • Install:

  • Conda
  • Pip
  • Source
pip install dask[complete]

git clone git+https://github.com/dask/dask
cd dask
pip install -e .

conda install dask

  • Deploy:

  • Local
  • CLI
  • YARN
  • Kubernetes
  • Helm
  • MPI
$ dask-scheduler
Starting scheduler at tcp://localhost:8786

$ dask-worker tcp://localhost:8786  # point workers to scheduler
$ dask-worker tcp://localhost:8786
$ dask-worker tcp://localhost:8786

$ pip install knit

>>> from knit.dask_yarn import YarnCluster
>>> cluster = YarnCluster(...)

$ pip install daskernetes

>>> from daskernetes import KubeCluster
>>> cluster = KubeCluster.from_yaml('worker-template.yaml')

$ helm repo add dask https://dask.github.io/helm-chart
$ helm repo update
$ helm install dask/dask

$ mpirun --np 10 dask-mpi
